Dictionary React App

This project was created as final project of SheCodes React Course.
It was implemented with CSS, HTML and React.js. I get data that is displayed on the page as a JSON file using AXIOS by API from the Free Dictionary API and I get pictures from Pexels.
On this website you can find meaning of the word, its synonyms, phonetic, examples of using and pictures that illustrate the word you are looking for. Also I created responsive design for better app look on mobile devices.

Kharkiv Landing Page

This project was created like landing page for SheCodes Challage.
On the page you can find facts about Kharkiv, as well as beautiful photos, but if you click on the last button in the navigation, then all the content will change and you will see how Kharkiv looks like because of the war, as well as introduce terrible facts.
This project was implemented with HTML, CSS and JavaScript. Also I created responsive design for better app look on mobile devices.

Weather React App

This site is a fully functional weather forecast search engine. Here you can find the weather in any city or by your geolocation. You can get acquainted with the weather data and see the weather forecast for two days ahead. The background image changes depending on the weather conditions.
It was implemented with CSS, HTML and React.js. I get data that is displayed on the page as a JSON file using AXIOS by API from the OpenWeather. Also I created responsive design for better app look on mobile devices.
