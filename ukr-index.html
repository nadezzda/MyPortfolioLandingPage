Проекти

Словник (React аpp)

Цей проект був створений як фінальний проект курсу SheCodes React.
Він був реалізований за допомогою CSS, HTML та React.js. Я отримую дані, які відображаються на сторінці, як файл JSON за допомогою AXIOS через API, з сайту Free Dictionary API, а також я отримую зображення з сайту Pexels.
На цьому веб-сайті ви можете знайти значення слова, його синоніми, фонетику, приклади вживання та картинки, які ілюструють потрібне слово. Також я створила адаптивний дизайн для кращого перегляду програми на мобільних пристроях.

Харків (Landing рage)

Цей проект був створений як лендінгова сторінка для SheCodes Challage.
На сторінці ви можете знайти факти про Харків, а також гарні фотографії, але якщо ви натиснете на останню кнопку в навігації, то весь вміст зміниться і ви побачите, як виглядає Харків через війну, а також ознайомитесь із жахливими фактами.
Цей проект реалізовано за допомогою HTML, CSS та JavaScript. Також я створила адаптивний дизайн для кращого перегляду програми на мобільних пристроях.

Прогноз погоди (React аpp)

Цей сайт є повнофункціональною пошуковою системою прогнозу погоди. Тут ви можете дізнатися погоду в будь-якому місті або за геолокацією. Ви можете ознайомитися з даними про погоду та подивитися прогноз погоди на два дні вперед. Фонове зображення змінюється в залежності від погодних умов.
Сайт був реалізований за допомогою CSS, HTML та React.js. Я отримую дані, які відображаються на сторінці як файл JSON за допомогою AXIOS через API з сайту OpenWeather. Також я створила адаптивний дизайн для кращого перегляду програми мобільні пристрої.
